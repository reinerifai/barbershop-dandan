# Barbershop Dandan

This repository is published as a GitHub Pages site.

Open the website here:

➡️ https://reinerifai.github.io/barbershop-dandan/

If you want the site to be published (so the link above works):

1. Go to the repository Pages settings: https://github.com/reinerifai/barbershop-dandan/settings/pages
2. Under "Source" choose the branch `main` and folder `/ (root)`, then Save.
3. Optionally set the **Homepage** field in Settings → General → Repository details to the URL above to show a prominent "Website" link on the repo header.

Note: GitHub does not let a repository URL (https://github.com/reinerifai/barbershop-dandan) automatically redirect to the Pages URL. The repository page will always show the code view. The best alternatives are:

- Enable Pages (steps above) and use the shown Pages URL or the Website link on the repo.
- Keep this README with a prominent link to the live site (just added).
